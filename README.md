  ## 2D Platformer Prototype

  Небольшой 2D-платформер, разработанный на Unity с использованием C#.
  Проект демонстрирует реализацию собственного контроллера персонажа, процедурную генерацию уровня и базовую игровую логику с системой прогресса.

  Основные механики:
  1) Плавное горизонтальное передвижение персонажа
  2) Прыжок с корректной обработкой состояния нахождения на земле
  3) Ограничение движения при контакте с препятствиями
  4) Система сбора монет
  5) Подсчёт дистанции, рекордов и монет
  6) Обработка состояния Game Over
  7) Интеграция анимаций через Animator

  Основные компоненты

1) RunnerController — управление персонажем
2) GameManager — управление состоянием игры
3) LevelGenerator — динамическая генерация уровня
4) CoinSpawner — генерация объектов на уровне
5) Coin — логика сбора монет
